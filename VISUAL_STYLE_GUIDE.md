# AARGO Clear Face Serum - Visual Style Guide

## SECTION 1: COLOR PALETTE

### Primary Colors (Foundation)

**White / Off-White (Background)**
- HEX: #FFFFFF (pure white) or #FAFAF8 (off-white with warmth)
- Usage: Main background for all sections, clean canvas for content
- Psychology: Trust, clarity, simplicity, premium feel

**Dark Charcoal (Primary Text)**
- HEX: #1A1A1A
- Usage: Body copy, headlines, primary text
- Psychology: High readability, professional, trusted
- Alternative: #2C2C2C for slightly softer feel

**Light Gray (Secondary Text)**
- HEX: #666666
- Usage: Captions, secondary information, metadata
- Psychology: De-emphasizes non-essential information

### Secondary Colors (Brand Identity - Inspired by Ingredients)

**Fresh Green (Green Tea, Aloe Vera - Calming, Natural)**
- HEX: #2D5F3F (dark forest green)
- Alternative: #4CAF50 (vibrant natural green)
- Usage: Accent sections, benefit cards, icons representing natural ingredients, trust badges
- Psychology: Growth, health, botanical, calming, safe

**Soft Blue (Hydration, Hyaluronic Acid)**
- HEX: #5BA3C7 (muted sky blue)
- Usage: Secondary accents, highlighting hydration benefits, calm sections
- Psychology: Hydration, cooling, trust, serenity

**Warm Coral (Vitamin C, Rosehip, Energy, Action)**
- HEX: #FF6B6B (warm coral-red)
- Alternative: #FB923C (warm orange-coral)
- Usage: Primary CTA buttons, urgency elements, call-to-action highlights
- Psychology: Energy, urgency, warmth, results, action-oriented

### Accent Colors (Supporting Elements)

**Soft Gold (Radiance, Premium Feel)**
- HEX: #D4AF37 (luxe gold)
- Alternative: #F9D866 (soft golden yellow)
- Usage: "Results" or "glow" icons, premium badges, success states
- Psychology: Premium, luxury, radiance, achievement

**Light Peach (Gentle, Approachable)**
- HEX: #FFE8D6 (warm peach)
- Usage: Soft background sections, gentle CTAs, calming transitions
- Psychology: Warmth, approachability, gentleness

**Sage Green (Balanced, Harmonious)**
- HEX: #A8B8A8 (muted sage)
- Usage: Secondary accent, balance/harmony icons, complementary to fresh green
- Psychology: Balance, harmony, natural, sophisticated

### Neutral Colors (Supporting)

**Light Gray Background**
- HEX: #F5F5F5 or #F0F0F0
- Usage: Section backgrounds, card backgrounds, alternating sections
- Psychology: Visual breathing room, organization

**Border Gray**
- HEX: #E0E0E0
- Usage: Subtle dividers, card borders, form inputs
- Psychology: Definition without heaviness

## SECTION 3: TYPOGRAPHY

### Font Stack (Recommended)

**Headings (Display Font)**
- Primary: **Montserrat Bold** (Google Fonts) – modern, geometric, confident
- Fallback: **Poppins Bold** (Google Fonts) – friendly, rounded, clean
- System fallback: `-apple-system, BlinkMacSystemFont, "Segoe UI", sans-serif`

**Body Text (Readable Font)**
- Primary: **Inter** (Google Fonts) – highly readable, neutral, professional
- Fallback: **Segoe UI** (system font) – excellent readability on Windows
- System fallback: `Roboto, "Helvetica Neue", Arial, sans-serif`

**Button Text**
- Font: **Poppins SemiBold** (Google Fonts)
- Fallback: **Montserrat SemiBold**
- System fallback: `-apple-system, BlinkMacSystemFont, "Segoe UI", sans-serif`

### Font Sizes & Hierarchy

#### Desktop / Tablet

| Element | Size | Weight | Line Height | Letter Spacing | Example |
|---------|------|--------|-------------|------------------|---------|
| **H1 (Hero Headline)** | 48-56px | Bold (700) | 1.2 | -0.5px | "Dark Spots Won't Fade? This Daily Serum..." |
| **H2 (Section Headlines)** | 32-40px | Bold (700) | 1.3 | -0.3px | "Meet Your New Skincare Essential" |
| **H3 (Sub-Headlines)** | 24-28px | SemiBold (600) | 1.4 | 0px | "15% Vitamin C + 3% Kojic Acid" |
| **H4 (Card Headlines)** | 18-20px | SemiBold (600) | 1.5 | 0px | Testimonial name, FAQ questions |
| **Body (Primary)** | 16-18px | Regular (400) | 1.6 | 0px | Main body copy, descriptions |
| **Body (Secondary)** | 14-16px | Regular (400) | 1.5 | 0px | Captions, metadata, smaller text |
| **Button Text** | 16-18px | SemiBold (600) | 1.5 | +0.5px | "Add to Cart", CTA buttons |
| **Small Text** | 12-14px | Regular (400) | 1.4 | 0px | Footer, fine print, disclaimers |

#### Mobile (320px - 480px)

| Element | Size | Weight | Line Height | Letter Spacing |
|---------|------|--------|-------------|------------------|
| **H1 (Hero Headline)** | 28-32px | Bold (700) | 1.3 | -0.3px |
| **H2 (Section Headlines)** | 22-26px | Bold (700) | 1.4 | -0.2px |

| **Small Text** | 12px | Regular (400) | 1.4 | 0px |

### Typography Best Practices

- **Contrast:** Ensure WCAG AA compliance (4.5:1 minimum for body text)
- **Line Length:** 50-75 characters per line (desktop), optimize for mobile
- **Hierarchy:** Use size, weight, and color to guide visual hierarchy
- **Readability:** Minimum 16px for body text on mobile for accessibility
- **Spacing:** Use generous line-height (1.5-1.8) for body copy to increase readability

## SECTION 4: BUTTON STYLES & CTA DESIGN

### Primary CTA Button (High Conversion Focus)

**Desktop & Tablet:**
- **Width:** 100% on mobile, 300-400px on desktop
- **Height:** 56px
- **Padding:** 16px vertical | 24px horizontal
- **Background Color:** #FF6B6B (warm coral)
- **Text Color:** #FFFFFF (white)
- **Font:** Poppins SemiBold, 18px
- **Letter Spacing:** +0.5px
- **Border:** None
- **Border Radius:** 4px (subtle) or 8px (modern)
- **Box Shadow:** 0 4px 12px rgba(255, 107, 107, 0.25) (subtle shadow)
- **Cursor:** pointer

**Hover State:**
- **Background Color:** #E55555 (darker coral)
- **Box Shadow:** 0 6px 16px rgba(255, 107, 107, 0.35) (deeper shadow)
- **Transform:** translateY(-2px) (subtle lift)
- **Transition:** all 0.2s ease-in-out

**Active/Pressed State:**
- **Background Color:** #C94D4D (darkest coral)
- **Box Shadow:** 0 2px 8px rgba(255, 107, 107, 0.2) (pressed effect)
- **Transform:** translateY(0px) (no lift)

**Disabled State:**
- **Background Color:** #CCCCCC (light gray)
- **Text Color:** #999999 (medium gray)
- **Cursor:** not-allowed
- **Opacity:** 0.6

**Mobile:**
- **Full-width:** 100% of container with 16px margin on each side
- **Height:** 56px (thumb-friendly, larger than desktop)
- **Tap Target:** Minimum 44px × 44px (accessibility standard)

### CSS Example
```css
.btn-primary {
  display: inline-flex;
  align-items: center;
  justify-content: center;
  width: 100%;
  height: 56px;
  padding: 16px 24px;

  color: #FFFFFF;

  background-color: #E55555;

  .btn-primary {

  }

```

## SECTION 5: SPACING & LAYOUT GUIDELINES

### Mobile-First Layout (320px - 480px)

**Margin/Padding Standards:**
- **Container Padding:** 16px left/right (outer margins)
- **Section Padding:** 24px top/bottom
- **Card Padding:** 16px
- **Element Spacing:** 12px - 16px (between elements within a section)
- **Section Spacing:** 40px - 60px (between major sections)

**Full-Width Elements:**
- Hero image: Full width, responsive height (16:9 or product-dependent)
- CTA buttons: Full width with 16px margins on each side
- Forms: Full width with 16px margins
- Benefit cards: Full width, stack vertically

**Column Layout:**
- Mobile: Single column, all elements stack
- Tablet (768px+): 2-3 columns for benefit cards, testimonials
- Desktop (1024px+): 2-4 columns, spacious layout

### Desktop Layout (1024px+)

**Container Width:**
- Max-width: 1200px
- Center on page with auto margins

**Spacing Standards:**
- **Container Padding:** 32px left/right (at desktop breakpoints)
- **Section Padding:** 48px top/bottom
- **Card Padding:** 24px
- **Element Spacing:** 16px - 24px
- **Section Spacing:** 60px - 80px

**Sticky Elements (Mobile):**
- Sticky "Add to Cart" button: Fixed at bottom (60px height)
- Appear on scroll (50px from top)
- Z-index: 1000 (above other content)
- Background: #FFFFFF with subtle shadow

### Grid System (Optional - for consistency)

- 12-column grid
- Mobile: 1 column (100% width - 2×16px margins)
- Tablet: 2-column or 3-column
- Desktop: 3-column or 4-column
- Gutter: 16px (mobile), 24px (tablet/desktop)

## SECTION 6: VISUAL HIERARCHY PRINCIPLES (Oli Gardner Framework)

### Attention Ratio Guide

**Primary Elements (First to Last):**
1. **Hero Image + Headline** (40% of initial attention)
2. **Primary CTA Button** (30% of initial attention)
3. **Sub-headline / Key Benefits** (20% of initial attention)
4. **Supporting Copy** (10% of initial attention)

**Elements Should Direct Eye Flow:**
- Large hero image at top center
- Headline directly below or overlaid on image
- Primary CTA immediately following headline
- Social proof section with before/after or testimonials
- Ingredient breakdown / solution section
- FAQ (supporting details)
- Final CTA (reiterate primary action)

**Design Principles:**
- Remove clutter: No unnecessary navigation, sidebars, or pop-ups
- Use whitespace strategically to emphasize key sections
- Color contrast to draw attention to CTAs and key benefits
- Size variation to create hierarchy (larger = more important)
- Font weight variation (bold for headlines, regular for body)

## SECTION 7: MOBILE OPTIMIZATION

### Mobile-First Approach

**Design Decisions:**
- Single-column layout (all elements stack vertically)
- Full-width images and buttons
- Simplified navigation (hamburger menu if needed)
- Thumb-friendly tap targets (minimum 44px × 44px)
- Reduced image sizes (WebP + JPEG fallback, <100KB per image)
- Compressed CSS/JavaScript

### Responsive Breakpoints

```css
/* Mobile (320px - 480px) */
@media (max-width: 480px) {
  /* Single column, large text, full-width elements */
}

/* Tablet (481px - 768px) */
@media (min-width: 481px) and (max-width: 768px) {
  /* 2-column layout, medium text, 50% width cards */
}

/* Desktop (769px - 1024px) */
@media (min-width: 769px) and (max-width: 1024px) {
  /* 3-column layout, normal text, wider containers */
}

/* Large Desktop (1025px+) */
@media (min-width: 1025px) {
  /* 4-column layout, comfortable spacing, max-width containers */
}
```

### Touch Optimization

- **Button Padding:** Minimum 56px height, 16px padding (for comfort)
- **Form Inputs:** Minimum 56px height, 16px padding (for thumbs)
- **Tap Targets:** Minimum 44px × 44px (accessibility standard)
- **Link Spacing:** At least 8px margin between interactive elements
- **Hover States:** Subtle, non-intrusive (color change, shadow, slight scale)

## SECTION 9: IMAGE GUIDELINES & PHOTOGRAPHY STYLE

### Product Photography

**AARGO Bottle Shots:**
- **Background:** Pure white (#FFFFFF) or very soft off-white (#FAFAF8)
- **Lighting:** Professional product lighting (3-point setup recommended)
- **Angle:** Straight-on (center-facing) for hero shot; 45-degree angle for lifestyle shots
- **Shadows:** Subtle drop shadow (2-4px) for definition
- **File Format:** WebP (primary) + JPEG (fallback)
- **Resolution:** 1200px × 1200px (minimum)
- **File Size:** <100KB (compressed for web)

**Texture Close-Ups:**
- Show lightweight, non-sticky texture
- Serum consistency close-up on white or clear background
- Resolution: 800px × 600px
- File Size: <50KB

### Before/After Imagery

**Style:**
- Real customer photos (diverse skin tones, genders, ages)
- Side-by-side or top-bottom layout
- Before (left) vs. After (right) or Before (top) vs. After (bottom)
- Consistent lighting and angle between before/after
- Focus on dark spots, tone evening, radiance

**Ethics & Authenticity:**
- Use real customer photos (with permission)
- No heavy filtering or Photoshop manipulation
- Disclose any professional editing (brightening, contrast adjustment only, no skin smoothing)
- Include "Actual customer result" label

**Resolution & Format:**
- 1200px × 900px (landscape) or 900px × 1200px (portrait)
- WebP (primary) + JPEG (fallback)
- File Size: <100KB each

### Lifestyle Photography

**Purpose:** Show real people using AARGO, diverse representation, approachable, aspirational

**Style:**
- Diverse Indian skin tones and ethnicities
- Various age ranges (18-50)
- Mix of gender representations
- Natural lighting (outdoor or bright window light)
- Casual, non-staged feel (but professionally shot)

**Angles & Compositions:**
- Close-up of face applying serum (45-degree angle, natural light)
- Overhead flat-lay of bottle + simple skincare routine
- Person holding bottle (lifestyle shot, arms visible, confident expression)
- Mirror shot of person inspecting skin (before/after transformation)

**Color Correction:**
- Warm, natural skin tones (avoid desaturating or cooling)
- Consistent color grading across all lifestyle shots
- Slight increase in vibrancy to convey freshness and results

**File Specs:**
- Resolution: 1200px × 1200px or 1200px × 1600px
- WebP (primary) + JPEG (fallback)
- File Size: <100KB each
- Alt Text: Descriptive, including skin type and benefit (e.g., "Woman with combination skin applying AARGO serum to cheek with visible dark spots fading")

### Ingredient Photography

**Purpose:** Build trust and educate on formula

**Style:**
- High-quality, professional shots of each key ingredient
- Isolated on neutral background (white or soft background)
- Artistic, beautiful presentation (macro photography if available)
- Warm, natural lighting

**Key Ingredients to Photograph:**
1. Vitamin C (crystal or powder form)
2. Kojic Acid (crystal or refined form)
3. Niacinamide (refined form)
4. Hyaluronic Acid (gel or transparent form)
5. Green Tea (leaves or brewed)
6. Aloe Vera (fresh plant or gel)
7. Rosehip Oil (in glass bottle, golden color)

**File Specs:**
- Resolution: 800px × 800px
- WebP (primary) + JPEG (fallback)
- File Size: <50KB each

### Graphics & Illustrations

**Purpose:** Visual interest, explainer content, reduce reliance on photography

**Style Options:**
- Option 1: Minimal line drawings (sketch-style, clean lines)
- Option 2: Flat design (geometric shapes, solid colors matching brand palette)
- Option 3: Semi-realistic (3D renders, professional but approachable)

**Recommended:** Flat design (consistent with modern skincare brands)

**Color Palette:** Match primary brand colors (#2D5F3F, #5BA3C7, #FF6B6B, #F9D866)

**Key Graphics:**
- Skin journey timeline (Week 0 → Week 4 → Week 8)
- Ingredient breakdown (icons + percentages)
- Routine comparison (3 serums vs. 1 serum)
- Before/after transformation visual

## SECTION 12: ACCESSIBILITY & WCAG Compliance

### Color Contrast Requirements

- **Normal Text:** Minimum 4.5:1 contrast ratio (WCAG AA)
- **Large Text (18pt+):** Minimum 3:1 contrast ratio
- **Interactive Elements:** Minimum 4.5:1 contrast ratio

**Tested Combinations:**
- Charcoal (#1A1A1A) on white (#FFFFFF): 18.54:1 ✓ (excellent)
- Light gray (#666666) on white (#FFFFFF): 5.12:1 ✓ (good)
- Coral (#FF6B6B) on white (#FFFFFF): 4.47:1 ✓ (acceptable; test with users)
- White text on green (#2D5F3F): 10.25:1 ✓ (excellent)
- White text on coral (#FF6B6B): 5.87:1 ✓ (good)

### Focus States & Keyboard Navigation

- All interactive elements must be keyboard-accessible (Tab key)
- Focus indicators: 2px solid outline (#5BA3C7 soft blue)
- Focus outline offset: 2px from element border
- No "outline: none" without providing alternative focus indicator

### Alt Text for Images

- Hero image: "AARGO Lifestyle Clear Face Serum bottle on clean white background"
- Before/after images: "Woman with dark spots on cheek before AARGO treatment | Same woman showing visibly lighter spots after 6 weeks of AARGO use"
- Ingredient photos: "Vitamin C crystal ingredient in AARGO Clear Face Serum"
- Decorative icons: Empty alt attribute (alt="")

### Form Accessibility

- All form inputs must have associated labels
- Error messages must be clearly visible (color + text)
- Required fields marked with asterisk (*) and aria-required="true"
- Form validation messages must be descriptive

## SECTION 13: PERFORMANCE OPTIMIZATION

### Image Optimization

**Format Strategy:**
- Primary: WebP (modern browsers, best compression)
- Fallback: JPEG (older browsers)
- Optimal compression: 75-80% quality

**Size Strategy:**
- Hero image: 1200px width, <100KB
- Benefit cards: 400px width, <40KB
- Testimonial photos: 150px × 150px, <20KB
- Ingredient photos: 400px × 400px, <50KB

**Lazy Loading:**
- Apply `loading="lazy"` to all below-the-fold images
- Use `srcset` for responsive images:
```html
<img 
  src="image-600w.webp" 
  srcset="image-600w.webp 600w, image-1200w.webp 1200w"
  sizes="(max-width: 600px) 100vw, 1200px"
  alt="Description"
  loading="lazy"
/>
```

### CSS & JavaScript Optimization

- Minify and compress CSS
- Defer non-critical JavaScript
- Use CSS Grid/Flexbox instead of floats
- Avoid unnecessary animations; prioritize 60fps

### Page Load Target

- **Target:** <3 seconds on 4G mobile connection
- **Measurement:** Use Google PageSpeed Insights, WebPageTest
